This is a test version of the Web Application made, as a solution to Pakathon: Aman Foundation and Civil Hospital Challenge.
Please, fill this form and provide a valid email address(at the end of the page) to send/receive a patient alert.

Although this is a minimal design, all tables are crucial and designed to communicate summary of medical history and calculate 7 important triage/prognosis scores. However, if there is a lack of time and/or user wishes to communicate some specific information only, any number of fields can be skipped(without affecting submission process).

Abbreviated Injurt Scoring System (AIS) is a common classification tool for injury classification. Although, calculating exact AIS is complicated , this App uses the following simple scheme. This is an important step in calculating Injury Severity Score(ISS). Use appropriate region and drop down menu to classify injury from Maximum to Minor or leave blank if not applicable to that particular anatomical region.
